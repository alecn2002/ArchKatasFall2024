# Architectural Katas: Fall 2024
This repository is the submissiion to [Architectural katas: Fall 2024](https://www.oreilly.com/live-events/architectural-katas-fall-2024/0642572006974/) by team [I2C](I2C_Team.md). 

The team proposes a rough first draft of an architecture for the **ClearView** System (aka "System") according to the [requirements](Diversity Cyber Council Kata Requirements 2024.md) proposed by the [Diversity Cyber Council](Diversity Cyber Council Kata Requirements 2024.md).
